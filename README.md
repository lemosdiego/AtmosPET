# AtmosPet

AtmosPet é uma plataforma dedicada à adoção de animais, permitindo que usuários encontrem pets disponíveis e compartilhem suas histórias de adoção. O site é intuitivo e responsivo, proporcionando uma experiência agradável para os visitantes.

## Sobre

AtmosPet é uma aplicação voltada para promover a adoção de animais, conectando pessoas que desejam adotar um pet a abrigos e ONGs que cuidam de animais. Além disso, a plataforma permite que os usuários compartilhem suas histórias de adoção, incentivando mais pessoas a adotarem.

## Funcionalidades

- Cadastro de usuários para login e registro.
- Visualização de pets disponíveis para adoção.
- Seção de histórias de adoção inspiradoras.
- Seção de perguntas frequentes e guia de adoção.
- Navegação fácil e intuitiva.

## Tecnologias Utilizadas

- **HTML**: Estruturação do conteúdo.
- **CSS**: Estilização das páginas.
- **JavaScript**: Interatividade e funcionalidades dinâmicas.
